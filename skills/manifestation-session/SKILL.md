# Manifestation Session

Run a short, grounded session that takes the user from a fuzzy want to a vivid, felt vision and one real action. The mechanism is mental rehearsal (which measurably changes behaviour) plus a specific intention (which biases attention) — not the universe delivering on request. Keep the tone of a calm guide, never a guru. Never promise outcomes; you set up the practice, the user lives it.

## When to use

The user asks to manifest something, visualize a goal, do an intention or vision session, or "get aligned" with what they want.

## Steps

1. **Get specific.** Ask for the one outcome they want, then tighten it. Vague ("more success") becomes concrete ("sign my first three coaching clients by March"). Specificity is load-bearing — reflect a sharpened one-sentence version back and confirm it.

2. **Find the felt detail.** Ask: when this is real, where are you, what do you see and hear, who's there, and what does the moment it lands *feel* like in the body? Collect 3–5 sensory anchors. The feeling is the lever, not the thinking.

3. **Guide the rehearsal.** Write a 1–2 minute first-person, present-tense script of that moment using their anchors. Tell them to read it slowly, eyes closed after, and run it like a film for a few minutes. This is the imagined-practice rep.

4. **Surface the obstacle (honesty step).** Ask for the single biggest real obstacle between now and the outcome. Outcome-only fantasy can *reduce* effort — pairing the wish with the obstacle (mental contrasting / WOOP) is what keeps it grounded.

5. **Commit one action.** Convert it into one implementation intention: "If [today's trigger], then I will [specific action]." Make it small enough to do before the day ends.

6. **Set up the log.** Remind them to note tonight what they did and anything related that surfaced (see the `evidence-log` skill).

## Output

Return: the sharpened one-sentence vision · the short rehearsal script · the named obstacle · the one "if-then" action for today. Keep it tight and usable.

## Boundaries

- Don't promise results or imply thinking alone delivers them. The action and the loop do the work.
- Don't frame anything as quantum physics or cosmic law. If the user wants that framing, gently keep it as metaphor.
- If the user raises distress, crisis, or clinical content, step out of the exercise and point them to real support (e.g. 988 in the US, or local services). This is a goals practice, not therapy.
